What is Dash?

Dash is a python framework created by plotly for creating interactive web applications. Dash is written on the top of Flask, Plotly.js and React.js. With Dash, you donâ€™t have to learn HTML, CSS and Javascript in order to create interactive dashboards, you only need python. Dash is open source and the application build using this framework are viewed on the web browser.

Execution steps

  1. Requirements to install have at least version 3.8 of python with the pip package manager.
  2. In the current dash server folder there should be three more files apart from this one.
  3. You only need to focus on the start.sh file, this executable can have two arguments the port and the host.
    4. sh start #default arguments port="8050" host="localhost" sh start -p 4000 #port 4000 default argument host="localhost" sh start -h 0.0.0.0 #host 0.0.0.0 default argument port="8050" sh start -h 0.0.0.0 -p 4000 #host 0.0.0.0 port 4000
  4. The program downloads the dependencies from the requirements file and then executes the app.py file.
  5. If everything went well you should see that the console is waiting and shows a web address.
